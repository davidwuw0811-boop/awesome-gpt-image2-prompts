- **分类筛选** — 快速定位所需类型的提示词

- **深色主题** — 科技感十足的现代化界面

## 💡 使用方法

### 1. 搜索提示词

在顶部搜索框输入关键词：
- 输入中文：如 "海报"、"3D"、"人物" 等
- 输入英文：如 "poster"、"3d"、"character" 等
- 支持模糊匹配和组合搜索

### 2. 按分类筛选

点击左侧或顶部的分类标签，快速定位特定类型的提示词。

### 3. 复制提示词

点击每张卡片上的 📋 复制按钮，即可将完整提示词复制到剪贴板。

### 4. 使用提示词

将复制的提示词粘贴到 GPT Image 2 中，根据需要进行修改和优化。
