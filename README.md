## 🚀 使用方法

1. 访问 [在线网页](https://davidwuw0811-boop.github.io/awesome-gpt-image2-prompts/)
2. 在搜索框输入关键词（支持中文和英文）
3. 点击分类标签进行筛选
4. 找到需要的提示词后，点击"复制"按钮
5. 粘贴到 GPT Image 2 中使用

## 📝 提示词格式

每条提示词包含以下信息：

- **中文标签** - 分类和描述，方便中文用户搜索
- **英文提示词** - 完整的英文提示词文本
- **作者** - 提示词的原作者
- **来源** - 数据来源仓库
- **参考图** - 部分提示词需要上传参考图片
